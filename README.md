# Система управления сметой

Приложение для управления строительными сметами.

## Запуск

1. Откройте файл `index.html` в браузере
2. Начните работу с добавления объектов

## Функционал

### Главная страница
- Просмотр списка всех объектов
- Добавление новых объектов
- Редактирование названий объектов
- Удаление объектов
- Открытие сметы объекта

### Страница сметы объекта

#### Вкладка "Бюджет"
- Добавление групп работ
- Добавление строк в группы
- Загрузка фотографий
- Выбор локации на карте (требуется API ключ Яндекс.Карт)
- Автоматический расчет сумм
- Итоги по группам и общий итог

#### Вкладка "Расход"
- Автоматическая синхронизация с планом
- Ввод фактических данных о закупках
- Дата покупки
- Фактические количество, цена и сумма
- Итоги по группам и общий итог

#### Вкладка "Анализ"
- Сравнение бюджета и расхода
- Отображение отклонений
- Процентное соотношение

## Примечания

- Все данные сохраняются в localStorage браузера
- Для работы карты необходим API ключ Яндекс.Карт (вставьте в object.html)
- Фотографии сохраняются в формате base64
